# IceFireDB-SQLite

IceFireDB SQLite数据库是一种去中心化SQLite数据库。提供一套方便的机制构建全球分布式数据库系统。 支持用户以MySQL协议向IceFireDB进行数据写入。 IceFireDB将数据存储到SQLite数据库，并将数据在P2P自动组网的节点间同步。
